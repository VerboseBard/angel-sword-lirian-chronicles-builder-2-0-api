# Ownership And Data Provenance

## Game Content

All Lyrian Chronicles / Angel's Sword game rules, names, text, artwork
references, and data in this project belong to the Angel's Sword TTRPG's
creators. The bundled rules data (`assets/lyrian-data.js`,
`assets/lyrian-detail-data.js`, `assets/versions/`, `data/angelssword/`) was
collected from their PUBLIC website and public API endpoints
(`rpg.angelssword.com`, `api.angelssword.com`) for the purpose of building a
fan-made character builder. No paid, private, or account-gated content is
included.

If the Angel's Sword team wants any of this data removed or replaced with an
official feed, that request wins — the app's data-provider layer exists
precisely so official data can replace the bundled copy.

## Application Code

The application code (builder, play sheet, wizards, dice system, import/export,
provider layer, tests, and docs) was created by this project's owner as a
fan work. It is shared with the Angel's Sword team for evaluation and
integration.

No open-source license has been chosen yet. If you need formal license terms
to evaluate or reuse this code, ask the project owner — do not assume terms.

## Secrets

This is a static browser app. Nothing in this repository is secret, and no
file in it may ever contain API keys, passwords, or tokens.
