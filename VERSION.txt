Lyrian Chronicles Character Builder - Beta Version 2.1 API Build
Promoted from the reviewed Beta 1.9 new-player downtime UX build.
Includes the Beta 1.6 disabled-by-default API provider handoff layer.
Includes a disabled-by-default security placeholder for future official backend/auth work.
Focus: preserve the 1.9 crafting and gathering workflows while preparing the build
for later official API integration without breaking static/local use.

1.9 fix sync (2026-07-07, after the handoff kit): ported the Beta 1.9
post-promotion fix session into this build - gathering outcome resolution
guard (no double-finish, resolved-state Resolve Gather step), breakthrough
EXP overflow into normal XP with Spirit Core accounting, compact locked
breakthrough cards with inspect-to-expand, per-type gathering node SVG icons,
clearer no-yield messaging, and three new functional regression tests
(gathering no-yield resolution, breakthrough general-XP spending, compact
locked card layout). Verified by 1.9's own suite before the port and this
build's full suite after it.

1.91 hotpatch sync (2026-07-08): ported the beta tester feedback hotpatch
from the live Beta 1.91 static build into this API build. Includes clearable
racial/builder choice controls, safer class-level hydration, corrected
ancestry inspection after race changes, equipment category/resource filters,
weapon proficiency hints, sticky remaining Clim while shopping, buyable
Food/Pine/Iron/Alchemy resource units, Food Units priced per unit with
+1/+5/+10/+50/+100 purchasing, and single-card Food Unit inventory
consume/add controls. Static/API behavior must remain unchanged.

API handoff kit (2026-07-07): local mock of the expected official endpoint
(/builder/game-data on the dev server), payload validator and sample payload,
API-mode + static-fallback regression tests, AI_INTEGRATION_TASK.md task brief,
docs/api-mode-ownership-map.md, and a data-mapper fix so valid detail data
(races/ancestries/classes shape) is no longer discarded in API mode.

2.1 feature sync (2026-07-15): updated the private API build with the complete
Standard/Mirane start system, campaign-aware pricing and crafting, retirement
and retraining review guidance, twelve optimized race-aware Quick Builds,
Slimefolk, Dullahan, Spiderfolk, newcomer class descriptions, and current
responsive/mobile tests.
The API provider, mock endpoint, validator, static fallback, and disabled
security placeholder remain intact and are covered by the full test suite.
